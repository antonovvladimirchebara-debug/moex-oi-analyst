ПЛЕЙЛИСТ AUDIO (ФАЙЛЫ + URL)

Несколько именованных плейлистов: загрузки с ПК и URL-потоки привязаны к выбранному плейлисту. «НА САЙТЕ ИГРАЕТ» — какой плейлист слушают посетители. «ЭФИР» — какие треки из него в ротации. MP3, WAV, OGG, FLAC, M4A до 50 МБ.

ДОБАВИТЬ ПО ССЫЛКЕ (ПОТОК)

Прямая HTTPS-ссылка на файл (mp3, ogg, m4a, wav…). Без CORS визуализатор FFT может не работать — воспроизведение обычно да.

ЯНДЕКС МУЗЫКА

Подключи свой аккаунт Яндекс Музыки через OAuth. Для этого нужно зарегистрировать приложение на oauth.yandex.ru и указать тип «Веб-сервис» с Callback URI = URL этой страницы.

♫ ЯНДЕКС МУЗЫКА НЕ ПОДКЛЮЧЁН

ПЛЕЙЛИСТ VIDEO (ФАЙЛЫ + URL)

Несколько именованных плейлистов: загрузки с ПК и URL-вставки привязаны к выбранному плейлисту. «НА САЙТЕ ИГРАЕТ» — какой плейлист видят посетители. «ЭФИР» — какие ролики из него в ротации. MP4, WebM, OGG, MOV до 80 МБ. YouTube, Vimeo, Rutube, VK, Dailymotion, прямые ссылки.

ДОБАВИТЬ ПО ССЫЛКЕ
